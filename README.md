# Cryptora

Cryptora is a web-based cybersecurity learning and practice platform built using Python Flask.

## Features
- User login and authentication
- Cybersecurity learning modules
- Quiz and assessment system
- Progress tracking
- Clean web interface

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
