🖥️ Operating System, Firewall, IDS & IPS

📌 Introduction

This section explains how Operating Systems manage hardware and software, and how cybersecurity tools like Firewalls, IDS, and IPS protect systems and networks from cyber threats.

🔸 What is an Operating System (OS)?

An Operating System is software that manages hardware and software resources on a computer. It provides a user interface, file system, process management, memory management, and device control.

🔸 Examples of OS

  • Windows
  • Linux
  • macOS
  • Android

🔥 Firewall – What is it?

A firewall is a network security system that monitors and controls incoming and outgoing network traffic based on security rules. It acts as a barrier between trusted and untrusted networks.

🛡️ IDS (Intrusion Detection System)

IDS is a system that monitors network or system activity for malicious actions or policy violations. It alerts administrators when a potential attack is detected.

🚫 IPS (Intrusion Prevention System)

IPS not only detects intrusions but also prevents them by blocking the suspicious traffic. It's like an upgraded version of IDS with automatic action capability.
