Cybersecurity Tools Knowledge Base 🛠️

Understand tools, their purpose, and basic commands.

🔎 Nmap

Purpose: Network scanning and port discovery.

Basic Commands:

  • nmap 192.168.1.1
  • nmap -sV target.com
  • nmap -A target.com
Download Tool

🕵️ Wireshark

Purpose: Capture and analyze network packets.

Filters:

  • http
  • ip.addr == 192.168.1.1
  • tcp.port == 80
Download Tool

🛡 Burp Suite

Purpose: Web vulnerability testing.

Usage: Intercept HTTP requests and test web apps.

  • Proxy → Intercept ON
  • Send to Repeater
  • Scan vulnerabilities
Download Tool

💣 Metasploit

Purpose: Exploitation framework for security testing.

Basic Commands:

  • msfconsole
  • search exploit
  • use exploit_name
